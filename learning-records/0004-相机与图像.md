## OCR 错误纠正

本教材对以下 OCR 识别错误进行了校正：

1. **原文 "P'_ ~ ~ ~ ~ ~ ~ ~ ~ Omega ~"** → 修正为成像点 $P'$（无 Omega 符号）
2. **原文 "u_{R^\odot}"** → 修正为 $u_R$
3. **原文 "Tmathfrak{D}"** → 修正为无意义符号，文中直接省略
4. **原文 "P_w )" 括号位置** → 修正为标准数学标注

## 关键公式卡片

- 针孔模型：$X' = fX/Z$, $Y' = fY/Z$
- 内参矩阵：$K = [f_x, 0, c_x; 0, f_y, c_y; 0, 0, 1]$
- 完整投影：$Z P_{uv} = K(R P_w + t)$
- 畸变纠正：$x_c = x(1 + k_1 r^2 + k_2 r^4 + k_3 r^6) + 2p_1 xy + p_2(r^2 + 2x^2)$
- 双目深度：$z = fb/d$
- 像素→3D：$x = (u-c_x)z/f_x$, $y = (v-c_y)z/f_y$
