## 核心收获

1. **状态估计的贝叶斯视角**：从后验概率 $P(\boldsymbol{x}_k | \boldsymbol{x}_0, \boldsymbol{u}_{1:k}, \boldsymbol{z}_{1:k})$ 出发，利用马尔可夫性拆解为运动模型和观测模型的递归形式。

2. **卡尔曼滤波器**：线性高斯系统下的最优无偏估计。预测（$\hat{\boldsymbol{x}}_{k|k-1} = \boldsymbol{A}_k \hat{\boldsymbol{x}}_{k-1} + \boldsymbol{B}_k \boldsymbol{u}_k$）+ 更新（$\boldsymbol{K}_k$ 加权融合观测）。5 条标准公式体系。

3. **EKF 的局限**：线性化误差（只在工作点一阶近似）、大存储量（每个路标保持协方差）、非高斯噪声不适用。这些限制推动了图优化的替代方案。

4. **BA 的稀疏结构**：每个误差项只关联一个相机和一个路标 → H 矩阵中 $\boldsymbol{H}_{11}$ 和 $\boldsymbol{H}_{22}$ 均为对角块矩阵，交叉块稀疏。

5. **Schur 消元（边缘化）**：利用 $\boldsymbol{C}$（路标块）的对角结构高效求逆，先消路标 → 求相机增量 → 回代路标增量。将求解复杂度从 $O((6m+3n)^3)$ 降至 $O(6m)^3$。

6. **鲁棒核函数**：Huber 核将大误差从二次增长切换为线性增长，抑制误匹配的影响。

7. **g2o 实践**：自定义 Vertex（顶点）+ Edge（边），通过 `setMarginalized(true)` 启用 Schur 消元，选用稀疏 Cholmod 求解器。

8. **Ceres 实践**：定义拟函数 `operator()` 实现自动求导，通过 `ParameterBlockOrdering` 灵活指定消元顺序。

## g2o vs Ceres 对比

| 维度 | g2o | Ceres |
|------|------|-------|
| 抽象模型 | 顶点 + 边（图优化） | 代价函数 + 残差块（Problem） |
| Schur 消元 | setMarginalized（必须消所有路标） | ParameterBlockOrdering（灵活） |
| 自动求导 | 需手动调用 linearizeOplus | 内置 AutoDiff（模板元） |
| 数据操作 | 复制到 Vertex/Edge → 优化 → 写回 | 直接操作原始数组 |
| 文档 | API 文档少 | API 文档详细 |
