# 使命：精读《视觉 SLAM 十四讲》

## 任务目标

本教学项目旨在帮助学习者**系统掌握视觉 SLAM 核心技术**，从理论到实践完整覆盖《视觉 SLAM 十四讲》全书的每一讲内容。

## 学习者背景

- **研究方向**：计算机视觉 / 移动机器人感知
- **已有基础**：LSeg 语言驱动的语义分割（已完成前 7 课）
- **学习方式**：按书中章节逐课精读，每课一个自包含的 HTML 教学页
- **学习要求**：深入浅出、公式用 LaTeX 渲染、原图嵌入、课后练习可折叠展开、识别并纠正 OCR 错误

## 课程信息

- **书名**：视觉 SLAM 十四讲：从理论到实践
- **作者**：高翔、张涛
- **核心问题**：如何在未知环境中，通过相机实现同步定位与地图构建（SLAM）
- **关键技术**：特征点法 / 直接法视觉里程计、后端优化（BA/位姿图）、回环检测、地图构建

## 课程规划（按书中章节）

1. **前言** — 本书讲什么、如何使用、风格约定、知识自测
2. **初识 SLAM** — SLAM 框架概述、各模块介绍、编程环境搭建
3. **三维空间刚体运动** — 旋转矩阵、四元数、欧拉角、Eigen 实践
4. **李群与李代数** — 群的定义、指数映射、扰动模型、Sophus 实践 ✅
5. **相机与图像** — 针孔模型、畸变、双目 / RGB-D 模型、图像表示、OpenCV 实践、点云拼接 ✅
6. **非线性优化** — 状态估计、最小二乘、Gauss-Newton / LM、Ceres / g2o
7. **视觉里程计 1（特征点法）** — 特征提取匹配、对极几何、PnP、ICP
8. **视觉里程计 2（直接法）** — 光流、直接法、RGB-D 直接法实践
9. **实践章：搭建 VO 框架** — 完整视觉里程计实现
10. **后端 1（BA 与稀疏性）** — Bundle Adjustment、稀疏求解加速
11. **后端 2（位姿图）** — 位姿图优化、g2o / gtsam 实践
12. **回环检测** — 词袋方法、字典训练、回环检测实践
13. **地图构建** — 单目稠密重建、RGB-D 点云 / 八叉树地图
14. **SLAM 开源项目与未来** — 现有项目分析、发展方向展望

## 成功标准

- 能够用手绘框图完整阐述一个视觉 SLAM 系统的模块组成与数据流
- 理解旋转矩阵、四元数、李代数等数学工具在 SLAM 中的作用
- 掌握特征点法与直接法的原理差异及适用场景
- 能够独立运行并调试书中提供的代码示例
- 能够阅读和理解最新的 SLAM 论文
