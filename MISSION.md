# Mission: 精读 E2Map 论文并准备组会报告

## 任务目标

本教学项目旨在帮助学习者**深入理解 E2Map 论文**，掌握其核心技术，并能够**在组会上用中文做一次完整的全流程汇报**。

## 学习者背景

- **研究方向**：移动机器人
- **汇报场景**：组会报告（听众为移动机器人方向的研究组成员）
- **汇报要求**：完整覆盖背景介绍、方法详解、实验分析

## 论文信息

- **标题**：E2Map: Experience-and-Emotion Map for Self-Reflective Robot Navigation with Language Models
- **核心问题**：如何在随机环境中让机器人利用自身经验实现"一次学习、立即调整"的行为修正
- **关键方法**：结合 LLM 知识 + 智能体真实经验的情绪地图（E2Map）
- **实验验证**：仿真（Gazebo）+ 真实世界场景

## 学习路径

1. 理解论文背景与动机 —— LLM 导航的现状与不足
2. 掌握方法核心 —— E2Map 构建、情绪建模与更新机制
3. 掌握系统流程 —— 从指令输入到行为调整的完整链路
4. 理解实验设计与结果分析
5. 学会组织组会报告的内容与讲解逻辑

## 成功标准

- 能够用自己的语言清晰解释 E2Map 的核心思想和技术细节
- 能够回答组会上可能提出的问题（为什么比基线好？情绪建模为什么有效？）
- 最终产出一份课件/笔记，可用于组会汇报
